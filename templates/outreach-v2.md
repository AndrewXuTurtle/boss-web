## Flow A: Initial Outreach (3 messages max)

### Message 1 — Hook (1-2 sentences)
```
Halo kak, nama saya dari BOSS Web 👋

Saya lihat [NAMA BISNIS] di [Instagram/Google Maps] — keren banget tempatnya! 
```

> Keep it SHORT. Reference something specific about their business.
> If they have good reviews: "rating 4.8 di Google Maps, impressed!"
> If they're on Instagram: "content Instagram-nya aktif banget, top!"

### Message 2 — The Offer (after they reply OR if no reply in 2h, send this)
```
Sebenernya saya bikinin website preview buat [NAMA BISNIS] sebagai gambaran:

🔗 [LINK_PREVIEW]

Ini cuma draft 30 menit — GRATIS lihat dulu, nggak ada komitmen.

Dari Rp 200rb doang. Hosting gratis 1 tahun. Deal baru bayar.
```

### Message 3 — Follow up (only if no reply in 1 day)
```
Kak, boleh saya tahu pendapatnya soal preview tadi?

Saya bisa bantu bikin versi final sesuai keinginan. Revisi sampai puas 🚀
```

## Handling Responses

### If they say "iya/tertarik"
```
Great! Ini langkah-langkahnya:
1. Saya finalize desain sesuai input dari kamu
2. Revisi sampai kamu puas (maksimal 3x)
3. Saya publish — website live dalam 2-3 hari
4. Bayar kalau sudah sesuai

Mau mulai dari mana? Desain warna, struktur halaman, atau fitur khusus?
```

### If they say "mahal" / "perlu pikirin"
```
Wajar kak, santai aja 🙏

Yang bikin saya semangat bantu bisnis lokal karena:
- Website = kredibilitas. Orang check bisnis di Google — kalau nggak punya website, credibility turun.
- Kompetitor kamu sudah punya website.
- Kami yang urus semua — kamu tinggal approve desain.

Tapi nggak ada pressure ya kak. Kalau ready, saya standby 🙌
```

### If they say "sudah punya website"
```
Wah sudah punya, bagus kak! 😄
Boleh tahu websitenya alamat apa?
(Biar saya tahu sudah seperti apa — kalau belum optimal masih bisa saya bantu upgrade)
```

### If they say "tidak tertarik" / "STOP"
```
Siap kak, saya hapus nomor Anda dari daftar. Mohon maaf kalau mengganggu 🙏
```

## Key Rules
1. **Send before 9 PM** — never at night
2. **No mass broadcast** — send one by one, personalized
3. **Never send before a reply** — Message 1 then WAIT for reply before Message 2
4. **Reference something REAL** — their Instagram post, review count, location
5. **Match their energy** — formal with "Bapak/Ibu", casual with "kak"
6. **Follow up ONCE only** — don't spam

## Lead Qualification (check BEFORE sending)
- [ ] They have an ACTIVE Instagram or Google Maps listing?
- [ ] Verified WhatsApp number (not a landline)?
- [ ] Physical business location?
- [ ] Category: restaurant/cafe/salon/clinic/hotel/klinik?
- [ ] No existing website? (check: theirname.com, theirname.id, search "site:theirname")

| Tue–Thu | 19:00–20:30 | After dinner, people relax |

## Stop Signals — Do NOT send to these businesses
- No Google Maps listing (can't verify existence)
- Landline only (WhatsApp not active)
- Very small warung with no social media (not ready to invest)
- Business that seems closed / inactive
