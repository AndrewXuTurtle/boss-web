# WhatsApp Outreach Templates — Web Dev Service

## Template A: Initial Outreach (Cold)
```
Halo [Nama/Toko]! 👋

Saya dari tim BOSS Web. Saya nemuin [toko/bisnis] [nama] lagi aktif banget di [Instagram/Tokopedia], 
tapi saya lihat belum punya website sendiri ya?

Nah, saya udah bikin contoh draft website buat [nama bisnis] sebagai gambaran: 
[LINK_DRAFT_SITE]

Ini cuma draft cepat — bisa diubah total sesuai keinginan. Kita punya 2 paket:

🔹 Simple (1 halaman landing, responsive) → Rp 200.000–300.000
🔹 Lengkap (multi halaman, profil + katalog + kontak) → Rp 400.000–500.000

Yang menarik: setelah jadi, websitenya bisa di-host GRATIS pake GitHub Pages, 
jadi nggak ada biaya bulanan. Domain .com bisa diurus terpisah.

Tertarik lihat-lihat? Bisa kita diskusi dulu tanpa kewajiban. 🙌
```

## Template B: Follow-up (3 days later)
```
Halo [Nama]! Masih ingat draft website yang saya kirim sebelumnya?

Saya cuma mau kasih tau — spesial untuk [bulan ini], saya kasih FREE 3 bulan 
maintenance untuk order pertama. 😊

Ada yang bisa saya bantu jelasin lebih lanjut? Santai aja.
```

## Template C: Close / Negotiate
```
Baik [Nama], untuk harga [200k/300k/400k/500k] sudah termasuk:

✅ Desain responsive (HP + laptop + tablet)
✅ Hosting GRATIS (GitHub Pages, nggak ada biaya berulang)
✅ 1x revisi desain
✅ Domain bisa diurus (optional, biaya domain sendiri ~150k/tahun)
✅ Gratis 3 bulan maintenance

Kalau deal, saya bisa selesai dalam 2-3 hari. Mau lanjut? 🚀
```

## Lead Qualification Checklist
- [ ] Bisnis ini aktif di sosial media? (ada engagement?)
- [ ] Google Maps listing? Ada website di listing?
- [ ] Punya toko fisik? Perlu web untuk jangkauan lebih luas?
- [ ] Produk/jasa mereka cocok dipasarkan online?
- [ ] Nomor WhatsApp terverifikasi?
