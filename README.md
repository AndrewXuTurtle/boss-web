# Lead Engine — Web Dev Pipeline

## Pricing
| Tier | Price | Scope |
|------|-------|-------|
| Simple | Rp 200.000–300.000 | 1-page landing, static, mobile-friendly, 1 product/brand |
| Complex | Rp 400.000–500.000 | Multi-page, product catalog, contact form, custom sections |

## Pipeline Stages
```
SCRAPE → QUALIFY → DRAFT SITE → OUTREACH → NEGOTIATE → BUILD → DELIVER
```

## How It Works
1. **Scrape** — Find UMKM/local businesses with digital gaps (Instagram-active but no website, Google Maps listing with broken site, etc.)
2. **Qualify** — Check if they'd genuinely benefit from a web presence
3. **Draft** — Build a quick demo site using real data (their name, products, etc.)
4. **Outreach** — Send WhatsApp with the demo link + professional pitch
5. **Negotiate** — Let them tweak & bid within the price band
6. **Build** — Finalize and deploy
7. **Deliver** — Handover + optional maintenance

## Files
- `scripts/scrape-leads.sh` — Lead finder
- `templates/outreach.md` — WhatsApp message templates
- `templates/site-template.html` — Quick site generator
- `pipeline/` — Lead tracking
- `leads/` — Saved leads
